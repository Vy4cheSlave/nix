## Конфигурация nix, darwin, home-manager, nix-homebrew на macOS

1. [Установка Nix](https://docs.determinate.systems/determinate-nix/#getting-started)
2. Скачиваем зип архив (на маках с этим бывают проблемы)
> [!worn] Возможно потребуется выполнить комманду `xcode-select --install`
3. Копируем цель `init` из Makefile в Терминал
4. Кайфуем
